# go_git_notebook_gui

- a git based notebook gui implement by go
- 使用golang制作的跨平台云笔记界面
	- 当前只能存储文字
	- 图片只能使用连接
- 支持网站
	- [GitHub](https://github.com/)
	- [Gitee](https://gitee.com/)
- 设置说明

## 注意

- markdown的文件关联只能使用绝对路径,包括图片显示
	- 相对路径因为程序和文件存储根目录不同的关系导致失效
